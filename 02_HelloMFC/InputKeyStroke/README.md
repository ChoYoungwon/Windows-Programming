### 코드 목적
키 누름 메시지 처리하기

### 주요 코드
- 키누름 메시지(`WM_KEYDOWN`, `WM_KEYUP`, `WM_SYSKEYDOWN`, `WM_SYSKEYUP`)에 따른 핸들링
- `OnSize()` : 윈도우 크기가 변할 때마다 발생하는 메시지
- `OnKeyDown()` : 키보드 메시지 핸들러, 방향키를 누를 때 마다 도형이 이동한다.
- 방향키를 눌러 위치값만 바꾼 후 `CWnd::Invalidate()` 함수를 호출하여 `WM_PAINT` 메시지가 발생하도록 한다.

### 키 누름 메시지의 단점
어느 키를 눌렀는지 판단할 때는 유용하지만, 누른 키에 해당하는 문자는 알기가 어렵다.  
-> 운영체제는 키 누름 메시지와 더불어 **문자 메시지**를 발생시킨다.
